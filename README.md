

Project 1: Parlay Calculator
This program helps you calculate the odds of your parlay and your potential profit. 
You can add as many odds as you want, and when you're done, you type "listo" and 
the program asks how much money you want to bet. Then it calculates and shows 
how much you would win.


Project 2: Penalty Shootout Simulator
Console game that simulates a 5-round penalty shootout against the computer. 
You choose where to shoot and where to save; the program keeps score for both 
sides and declares a win, loss, or draw.
